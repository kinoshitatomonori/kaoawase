# kaoawase - リアルタイムクイズ大会

イベント用のリアルタイムクイズ大会アプリ。URLを共有して参加者がクイズに回答し、管理者がリアルタイムで結果を確認できる。

## セットアップ

```bash
npm install
npm run dev
```

## 使い方

### 参加者
1. http://localhost:3000 を開く
2. 名前を入力して登録
3. クイズを順番に回答して提出

### 管理者
1. http://localhost:3000/admin を開く
2. 各問題の選択肢ごとの得票数をリアルタイムで確認
3. 上位3名のランキングを確認

## クイズ問題の変更

`data/questions.ts` を編集してください。

## DBリセット

`db/quiz.db` を削除して再起動するとデータがリセットされます。
